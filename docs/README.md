## Quick Reference

**Theme**: Dark-first design with dark backgrounds and light text

**Colors**:
- Backgrounds: Black (#000000) and dark grays (zinc-900, zinc-950)
- Text: White (#FFFFFF) for headlines, zinc-400 for body text
- Accents: Bright Blue (#0066FF) and Bright Green (#00D084)
- Gradients: Dark blue-purple gradients for depth and visual interest

**Fonts**:
- Primary: Geist Sans
- Mono: Geist Mono

**Style**: Modern dark minimal - clean, spacious, content-focused with dark theme

**Tone**: Approachable and clear

## Using These Guidelines

When building pages or components:

1. Start with [Visual Style Guide](./visual-style.md) for layout and component patterns
2. Reference [Color Palette](./color-palette.md) for color usage
3. Use [Typography](./typography.md) for text hierarchy
4. Follow [Voice and Tone](./voice-and-tone.md) when writing copy
5. Ensure alignment with [Brand Identity](./brand-identity.md) principles

## Maintenance

These guidelines should evolve with the brand. Update documentation when:
- Adding new colors or modifying the palette
- Introducing new component patterns
- Changing typography scale or fonts
- Refining voice and tone based on user feedback
